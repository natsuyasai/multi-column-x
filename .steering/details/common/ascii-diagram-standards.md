# ASCII図表標準

## 必須：基本ASCIIのみ使用

**重要**：図表には必ず基本ASCII文字を使用してください（最大互換性のため）。

### ✅ 使用可能：`+` `-` `|` `^` `v` `<` `>` および英数字テキスト

### ❌ 禁止：Unicodeボックス描画文字

- 不可：`┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 理由：フォント/プラットフォームによって描画が一貫しない

## 標準ASCII図表パターン

### 重要：文字幅ルール

**ボックス内のすべての行は、スペースを含めて厳密に同じ文字数でなければなりません**

✅ 正しい例（すべての行 = 67文字）：

```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 誤った例（幅が不一致）：

```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### ボックスパターン

```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |

```

## 検証

図表作成前に確認：

- [ ] 基本ASCIIのみ：`+` `-` `|` `^` `v` `<` `>`
- [ ] Unicodeボックス描画なし
- [ ] 配置にスペース使用（タブ不可）
- [ ] 角には`+`を使用
- [ ] **すべてのボックス行が同じ文字幅**（スペースを含む文字数をカウント）
- [ ] 確認：等幅フォントで角が垂直に揃っているか検証
